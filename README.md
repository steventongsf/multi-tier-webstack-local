# Local Multi-tier application demo

## Overview
Each of these services runs in a dedicated VM and the VMs are created and managed using Vagrant.  These services include:
* Nginx
* Tomcat
* RabbitMQ
* Memcached
* MySql


## Bring up environment
```
vagrant up
```
## Remove environment
```
vagrant destroy
```
