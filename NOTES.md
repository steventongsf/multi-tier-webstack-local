# Application Services
* nginx
* tomcat
* mysql
* memcached
* rabbitmq

# Local Setup
Vagrant >=2.3.3
VirtualBox >=7.0.2

# Runtime
## Start environment
```
vagrant plugin install vagrant-hostmanager
vagrant up
```
## Suspend/Resume env
```
vagrant suspend
vagrant resume

```

## Stop environment
```
vagrant destroy
```
